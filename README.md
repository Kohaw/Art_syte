# ИДЗ2-проект
АРТ-АКАДЕМИЯ — учебный проект сайта школы рисования для взролсых. Это адаптивный лендинг, разработанный в рамках практики HTML, CSS и JavaScript.

## Функциональность
- Адаптивная верстка (mobile / tablet / desktop)
- FAQ-аккордеон
- Бургер-меню
- Модальные окна
- Форма записи с валидацией
- Маска ввода телефона

## Технологии
- HTML
- CSS
- JavaScript

Учебный проект
